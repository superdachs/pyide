Welcome to Pyide

Pyide is a new Python based IDE for, what else, Python. Goal is a full featured IDE for creating complex Python projects with or without virtualenv. Also Django support will be added. In future releases there will also be certain deployment functionality for creating webapps and packages. Also support of glade GUI builder is planned. Therefore it will be connected with Glade GUI Designer or implement own Gui Builder.
